❓ 常见问题

以下是用户最常咨询的问题

RAILGUN 是免费软件吗？
+

是的，RAILGUN 是一款完全免费的桌面应用程序，所有核心功能均可免费使用。我们相信好的工具应该让每个人都能轻松获取。

软件支持哪些操作系统？
+

目前 RAILGUN 支持 Windows 10/11 操作系统。我们正在开发 macOS 和 Linux 版本，敬请期待！

如何下载音乐？
+

您可以使用 RAILGUN 内置的 B站下载功能来下载音乐视频。只需复制视频链接，粘贴到下载工具中即可开始下载。

在线工具需要安装客户端吗？
+

不需要！RAILGUN 在线工具网站完全基于浏览器运行，无需安装任何软件。只需打开网站即可使用所有工具。

我的数据会被保存吗？
+

您的数据安全对我们至关重要。我们不会未经授权收集或保存您的个人数据。具体隐私政策请参阅官网说明。

如何参与软件开发？
+

我们欢迎开发者参与 RAILGUN 的开发！您可以通过 GitHub 提交 Issue 或 Pull Request，也可以在联系页面加入我们的开发者交流群。
